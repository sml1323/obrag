## Animation Guidelines

### Micro-interactions

| Element | Trigger | Animation |
|---------|---------|-----------|
| Button | Hover | Lift -2px, shadow grows |
| Button | Click | Drop to 0, shadow shrinks |
| Card | Hover | Lift -2px, shadow grows |
| Tab | Switch | Slide indicator bar |
| Character | State change | Crossfade (0.3s) |
| Toast | Enter | Slide from top + bounce |
| Toast | Exit | Fade out |

### Page Transitions

| Transition | Animation |
|------------|-----------|
| Tab switch | Fade + slight slide (0.2s) |
| Modal open | Scale from 0.9 + fade (0.2s) |
| Modal close | Scale to 0.9 + fade (0.15s) |
